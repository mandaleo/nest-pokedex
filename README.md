# Execute in development

1. Clone repository
2. Execute

```
yarn install
```

3. Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Run database with Docker

```
docker-compose up -d
```

## Stack

- NestJS
- MongoDB
